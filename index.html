‚ùì FAQ (Pertanyaan Umum)

Kenapa tool ini aman dipakai?

Karena tool ini cuma ambil data via cookie resmi Instagram kamu sendiri, tanpa login ke server lain atau pihak ketiga. Semua proses berjalan lokal di HP kamu lewat Termux.

Kenapa cookie harus diambil manual?

Itu cara paling aman biar akun kamu gak keambil pihak lain. Ambil manual via Kiwi Browser pake extension EditThisCookie biar cookie-nya valid & resmi dari web IG kamu sendiri.

Apa itu sessionid dan ds_user_id?

Keduanya adalah identitas akun kamu di browser. Sessionid = login token, ds_user_id = ID akun kamu. Tool ini cuma butuh dua itu buat akses data followers/following kamu.

Kenapa muncul error 403 / 429?

Itu artinya Instagram lagi ngebatesin permintaan (rate limit). Tunggu 1‚Äì2 menit lalu jalankan ulang. Tool ini udah otomatis ada delay biar gak ke-ban IP.

Berapa lama cookie aktif?

Biasanya 30 hari. Kalau tool tiba-tiba gagal ambil data, ulangi aja langkah node setup-cookie.js buat update cookie baru.
